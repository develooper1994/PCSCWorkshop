## Key Architecture Decisions

- **No std::function** — use templates (DesfireAuth, DesfireCommands callbacks)
- **Result<T> pattern** — exception-free error handling
- **PCSC centralized** — single entry point (no CardConnection wrapper)
- **Platform.h guards** — `#ifdef _WIN32` for PCSC include/API differences
- **CNG Win-only** — cipher/reader platform-independent; CNG in conditional compile
